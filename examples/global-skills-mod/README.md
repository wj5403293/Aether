# Global Skills Mod

This example demonstrates Mod Kernel v2 by implementing the workflow requested
in Aether issue #27:

- replaces the native composer action tray;
- hides the native Skill picker;
- groups and collapses selected Skills;
- writes directly to Aether's native selected Skill IDs;
- persists global defaults for every new conversation.

Import the directory as a zip package through **Settings → Extensions**.
