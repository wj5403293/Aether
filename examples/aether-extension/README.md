# Combined Pi + Aether extension example

Zip this directory and import it from Aether's Extensions page.

The package contains:

- a standard Pi command (`/aether-example`);
- a native card above Aether's composer;
- an extension-owned native page in the conversation drawer;
- a WebView using the JavaScript action bridge;
- persistent extension state and Android host calls.

See `docs/AETHER_EXTENSIONS.md` for the complete API.
